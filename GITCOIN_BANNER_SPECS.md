# Gitcoin Banner Image Specifications

---

## Quick Requirements
- **Dimensions:** 1200 x 400 pixels
- **Format:** PNG or JPG
- **File size:** <2 MB (ideally <500 KB)
- **Design tool:** Canva (free), Figma, or Photoshop

---

## Design Guidelines

### Color Palette
**Primary:**
- Deep Green: #2C5F2D (renewable energy)
- Solar Gold: #F4A62A (sun/energy)
- Electric Blue: #00D4FF (tech/blockchain)

**Accent:**
- White: #FFFFFF (text)
- Dark Gray: #1A1A1A (background overlay)

### Layout Structure
```
┌────────────────────────────────────────────────────────────┐
│                                                            │
│  [ICON/LOGO]          SolarPunk Protocol                   │

│                                                            │
│              [Background: Solar panels + blockchain hex]   │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

## Canva Quick-Start (15 minutes)

1. **Open Canva** → Search "LinkedIn Banner" (similar dimensions)
2. **Resize** → Custom size: 1200 x 400 px
3. **Background:**
   - Elements → Search "gradient green blue"
   - Elements → Search "solar panels" (use as overlay at 30% opacity)
4. **Text:**
   - Add heading: "SolarPunk Protocol" (Montserrat Bold, 72pt)
   - Add subheading: "Decentralized Energy Derivatives" (Montserrat Regular, 28pt)
5. **Icons:**
   - Elements → Search "sun icon" (place left)
   - Elements → Search "blockchain" (place right)
6. **Export:** Download → PNG → High quality

## Quality Checklist

Before uploading to Gitcoin:
- [ ] Text is readable at 50% zoom (test legibility)
- [ ] Dimensions are exactly 1200 x 400 px
- [ ] File size < 2 MB
- [ ] Brand consistent with your GitHub README aesthetic
- [ ] No typos in text
- [ ] Colors have good contrast (use WebAIM contrast checker)
- [ ] Looks professional on both light and dark mode (Gitcoin uses both)

---

## Gitcoin Profile Tips

**Use the banner to:**
- Show professionalism (MVP ready, not just idea)
- Communicate value prop in 3 seconds
- Build trust (tests passing, technical credibility)
- Stand out (most projects have generic banners)

**What to avoid:**
- Cluttered design with too much text
- Low-resolution images (looks unprofessional)
- Generic blockchain stock photos (overused)
- Missing contrast (text blends into background)
